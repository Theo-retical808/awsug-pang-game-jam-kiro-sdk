# 🎮 AWSUG PANG Game Jam  
# Paint the Canvas Purple with Kiro

Welcome to the official repository of the  
## AWS User Group Pangasinan (AWSUG PANG) Game Jam

This event is a rapid AI-assisted development challenge focused on:
- creativity,
- software architecture,
- gameplay innovation,
- and effective AI usage.

Participants will transform a starter game template into something unique using only limited AI resources.

---

# 🟣 Theme: Paint the Canvas Purple with Kiro

Every contestant begins with the same blank foundation — a starter template.

Your mission is to:
> take a generic framework and transform it into your own creative masterpiece using Kiro IDE.

The phrase:
## “Paint the Canvas Purple”
represents:
- creativity,
- innovation,
- transformation,
- experimentation,
- and the unique identity of AWSUG PANG.

You are not expected to build a game completely from scratch.

Instead, you are challenged to:
- expand,
- redesign,
- improve,
- overhaul,
- and creatively evolve an existing template into something memorable.

---

# 🧠 Objective

Using only:
- Kiro IDE
- the provided game templates
- and your own creativity

Participants must:
1. Clone this repository
2. Choose ONE template
3. Improve or overhaul the template
4. Create unique gameplay or experiences
5. Stay under the 50-credit AI usage limit
6. Submit their finished project through GitHub
